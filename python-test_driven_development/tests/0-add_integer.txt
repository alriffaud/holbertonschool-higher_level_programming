=====================
Using ``add_integer``
=====================

First we import ``add_integer`` from the ``0-add_integer`` module:

    >>> add_integer = __import__('0-add_integer').add_integer

Now use it:

Numbers
=======

>>> add_integer(2, 3)
5

Non-Numbers
===========

>>> add_integer('a', 5)
Traceback (most recent call last):
...
TypeError: a must be an integer
